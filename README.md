#  Análisis de Datos con Pandas y NumPy

Este repositorio contiene ejercicios prácticos y un proyecto aplicado para el aprendizaje de análisis de datos utilizando Python, específicamente con las librerías Pandas y NumPy.

##  Objetivo
Desarrollar habilidades en manipulación, limpieza y análisis de datos, aplicando técnicas reales utilizadas en entornos profesionales.

##  Estructura del Proyecto
- `data/`: dataset de ventas en formato CSV
- `scripts/`: scripts de Python con el desarrollo del análisis
- `img/`: visualizaciones generadas a partir de los datos
- `notebooks/`: pruebas y experimentación

##  Análisis Realizado
Se desarrolló un análisis de ventas que incluye:

- Cálculo de ingresos totales (`total_venta`)
- Identificación de la ciudad con mayores ventas
- Análisis del producto más rentable
- Comparación entre volumen de ventas y generación de ingresos
- Cálculo de métricas como promedio y mediana
- Limpieza y transformación de datos
- Generación de gráficos:
  - Ventas por ciudad
  - Ventas por producto
  - Tendencia de ventas en el tiempo

##  Tecnologías Utilizadas
- Python
- Pandas
- NumPy
- Matplotlib

##  Conclusión
El proyecto permite entender cómo diferentes productos impactan el negocio de distintas formas (alto valor vs alto volumen), y cómo el análisis de datos puede apoyar la toma de decisiones.
